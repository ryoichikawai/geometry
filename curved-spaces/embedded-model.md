# Embedded models

[Nash embedding theorem](https://en.wikipedia.org/wiki/Nash_embedding_theorem) – Every Riemannian manifold can be isometrically embedded into some Euclidean space

[Whitney embedding theorem](https://en.wikipedia.org/wiki/Whitney_embedding_theorem) – Any smooth real m-dimensional manifold can be smoothly embedded in $\mathbb{R}^{2m}$.



## 2-sphere embedded in $\mathbb{R}^3$



```{prf:definition} 2-Sphere
:label: def-2-sphere

$S^2 = \{ x \in \mathbb{R}^3 : \|x\|=1\}
```
