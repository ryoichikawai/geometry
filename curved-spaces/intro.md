# What is non-Euclidean space?



In this chapter, we try to find out what are Euclidean  and non-Euclidean spaces, and their differences.

First of all, what is a space?   It can mean many things.   We are interested in _space_ used in mathematics, such as vector space and Hilbert space.  In mathematics, a space is a set with added structures such as _topology_, _metric_ and _algebra_.     We call the elements of a set _points_.  We assign some meanings to points and define some relations between  them.  Then we have a space.  For example, if elements of a set satisfy certain algebraic rules for additions and scalar multiplications, we have a space known as vector space and its elements are called vector.

In the following sections, we first review Euclidean space which we are intuitively familiar with.  Then, we move on to non-Euclidean space.  We will see many common senses we developed through our experience break down.
